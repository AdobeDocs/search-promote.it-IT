# Introduzione{#getting-started}

Se hai poca esperienza con Search&amp;Promote e la navigazione dinamica, inizia qui per iniziare a usare il tuo account. Tra le altre cose, imparerai come indicizzare il tuo sito web e personalizzare l&#39;aspetto dei tuoi risultati di ricerca.

## Configurazione dell&#39;account {#section_FD4C71A422BD4ACEA51AFC98CDD3211C}

Prima di iniziare a personalizzare il tuo account, modifica la password inizialmente assegnata a te.

Consulta [Modifica della password di accesso](c-about-settings-menu/c-about-my-profile-menu.md#task_F5FF13AAD1514FE997C8882D4537C0C9).

È inoltre necessario aggiungere o modificare le impostazioni personali, ad esempio la codifica del set di caratteri, le informazioni di contatto, l’indirizzo e-mail e le preferenze.

Consulta [Configurazione delle informazioni utente personali](c-about-settings-menu/c-about-my-profile-menu.md#task_A11A3BE2527B4204B896E04303B04AA6).

Consulta [Configurazione delle preferenze](c-about-settings-menu/c-about-my-profile-menu.md#task_5E06BF565C284C2EBBE18E10A1C4BFBB).

Una volta completate le impostazioni personali, apri le impostazioni a livello di account.

Consulta [Configurazione delle impostazioni account](c-about-settings-menu/c-about-account-options-menu.md#task_80A38D0C8E4F453395BD67B81E4B45D9).

Rivedi il nome dell’account e l’indirizzo del sito web, quindi seleziona un fuso orario da utilizzare per i rapporti e la pianificazione. L&#39;indirizzo del sito web inserito è il sito principale &quot;entrypoint&quot; per il tuo sito. Tutte le pagine sotto il punto di ingresso vengono sottoposte a ricerca per indicizzazione e indicizzate se sono collegate, direttamente o indirettamente, all’URL specificato. Ad esempio, se il punto di ingresso è `https://www.mysite.com/`, la pagina `https://www.mysite.com/news/04.html` è indicizzata, se è collegata dalla pagina principale `index.html` o da una pagina collegata da tale indice. Se il sito web contiene sezioni non collegate dal punto di ingresso principale, ad esempio pagine di un secondo dominio, puoi specificare altri punti di ingresso URL.

Consulta [Aggiunta di più punti di ingresso URL da indicizzare](c-about-settings-menu/c-about-crawling-menu.md#task_2338A47387D74CFDAC4D4EF4A367ED45).

## Aggiunta di codice modulo di ricerca di esempio alle pagine web {#section_2BC00BE7D81245188E4A89E09D41D387}

Ora puoi aggiungere il codice di ricerca alle tue pagine web. Viene fornito un codice modulo di esempio per un modulo di ricerca standard.

Vedere [Anteprima del modulo di ricerca così come apparirebbe sul proprio...](c-about-auto-complete.md#task_437B35EFA5424603A08AF8E79E6B4714).

Vedere [Copia del codice HTML del modulo di ricerca in...](c-about-auto-complete.md#task_A3A01EA800F24C0AA33902387E0362C7).

Copia e incolla il codice nelle pagine web in cui desideri visualizzare un modulo di ricerca. Ad esempio, molti siti web inseriscono un modulo nella propria home page o in un frame di navigazione. È possibile modificare il codice del modulo in base alle proprie esigenze di progettazione e contenuto oppure aggiungere ulteriori parametri di ricerca.

## Personalizzazione del collegamento dei risultati di ricerca {#section_0B4E2D6E1A3E4D3FBDBB607260CDB111}

Facoltativamente, puoi personalizzare il collegamento dei risultati di ricerca, i colori di sfondo e le informazioni di intestazione.

Consulta [Informazioni sui modelli](c-about-design-menu/c-about-templates.md#concept_06EB481B14864E18A8AE2BCD1D6EF0B5).

Vedere [Aggiunta di un nuovo file di presentazione o di modello di trasporto](c-about-design-menu/c-about-templates.md#task_73199757B6E748CAA604902FF913F012).

Vedere [Modifica di una presentazione o di un modello di trasporto](c-about-design-menu/c-about-templates.md#task_800E0E2265C34C028C92FEB5A1243EC3).

Quando sei soddisfatto dell’aspetto del modello, salva le modifiche e fai clic su **[!UICONTROL Push Live]**.

Vedere [Esecuzione di un indice completo di un sito web live o in staging...](c-about-index-menu/c-about-full-index.md#task_F7FE04D8A1654A7787FCCA31B45EB42D).

## Indicizzazione del sito Web {#section_6BBD16929CF04CB2A78BD657681C06B8}

Il sito web viene indicizzato quando crei un account per la prima volta. Tuttavia, se hai aggiunto nuovi contenuti o ne modifichi quelli esistenti, reindicizza il sito web.

Vedere [Esecuzione di un indice completo di un sito web live o in staging...](c-about-index-menu/c-about-full-index.md#task_F7FE04D8A1654A7787FCCA31B45EB42D).

È inoltre possibile pianificare l&#39;ora dell&#39;indice regolare.

Consulta [Impostazione della pianificazione completa dell&#39;indice per un sito web live](c-about-index-menu/c-about-full-index.md#task_6760F3256D004A228B38968DF15421F0).

Vedi [Esecuzione di un indice incrementale di un sito web live o in staging...](c-about-index-menu/c-about-incremental-index.md#task_9BFB6157F3884B2FAECB7E0E9CA318CB).

Se il sito web è grande o se si desidera reindicizzare solo un set di pagine modificate di frequente, è possibile configurare ed eseguire un indice incrementale.

Consulta [Configurazione di un indice incrementale di un sito web organizzato](c-about-index-menu/c-about-incremental-index.md#task_46A367B0786C4C90BFFA5D3F95FD86C0).

Vedi [Esecuzione di un indice incrementale di un sito web live o in staging...](c-about-index-menu/c-about-incremental-index.md#task_9BFB6157F3884B2FAECB7E0E9CA318CB).

Per eseguire un indice incrementale senza dover eseguire l&#39;accesso, è possibile creare un indice con script.

Vedere [Configurazione di un indice incrementale con script](c-about-index-menu/c-about-scripted-index.md#task_05AE040FE75E40FFAA5E10B6B6D4D255).

Oppure, puoi utilizzare uno script per trasmettere una richiesta remota di indicizzazione.

Vedere [Configurazione del telecomando per l&#39;indicizzazione](c-about-index-menu/c-about-remote-control-for-indexing.md#task_57C296258404448DA7A5ADC9B7232391).

## Visualizzazione dei rapporti sui termini di ricerca {#section_1F4A7F57B9C64FBDA5E3E592621BF6F4}

Le query di ricerca dei visitatori vengono registrate e i rapporti vengono creati per te per giorno, settimana e mese. Queste query ti consentono di vedere cosa cercano i clienti, quanto spesso hanno successo e dove la navigazione del tuo sito web si interrompe.

Consultare [Visualizzazione del rapporto Termini o dei termini di ricerca Null..](c-about-reports-menu/c-about-reports-menu.md#task_53B7ED1582DD4B0E8376546A7AFC789A).

## Attività avanzate per personalizzare l’esperienza di ricerca {#section_036286EC316F47C89D0D50C2D061BB59}

Ci sono molte opzioni aggiuntive che ti permettono di personalizzare e controllare completamente l&#39;esperienza di ricerca del cliente.

* Controlla quali parti del sito sono e non sono indicizzate con le maschere URL.

   Consulta [Informazioni sulle maschere URL](c-about-settings-menu/c-about-crawling-menu.md#concept_8039DFC53FF3410AA494D602F71BA164).
* Includi o escludi i file in base alle rispettive date.

   Consulta [Informazioni sulle maschere di data](c-about-settings-menu/c-about-crawling-menu.md#concept_F4F1F58A646F4A86B8650EC46FDCEF66).
* Se il sito web include documenti PDF o file con altri tipi di contenuto, ad esempio file MP3 o file Microsoft Office, è possibile scegliere di indicizzare o non indicizzare tali file.

   Consulta [Informazioni sui tipi di contenuto](c-about-settings-menu/c-about-crawling-menu.md#concept_6FEA1355C0374500B4C53090C34A8A07).
* È inoltre possibile fornire password in modo che la ricerca/merchandising del sito possa accedere alle aree protette da password del sito web per la ricerca.

   Consultare [Informazioni sulle password](c-about-settings-menu/c-about-crawling-menu.md#concept_3EDBD731725D46B891F834D4472774DC).
* Controlla il numero di connessioni utilizzate per indicizzare il tuo sito web.

   Vedere [Informazioni sulle connessioni](c-about-settings-menu/c-about-crawling-menu.md#concept_E2F3B7E7521147479E5948A94BB3A40B).
* Fornire informazioni sui moduli che si desidera sottoporre per indicizzazione e inviare.

   Consultare [Informazioni sull’invio di moduli](c-about-settings-menu/c-about-crawling-menu.md#concept_CADD5D7CF373497DAA6F8564D7BC8502).
* Escludere le parole comuni come &quot;a&quot; o &quot;the&quot; dalle ricerche.

   Consulta [Informazioni sulle parole escluse](c-about-linguistics-menu/c-about-excluded-words.md#concept_9DB67BD2F0DC43AC88741003D9F39812).
* Specifica alcune aree del tuo sito web da cercare, ad esempio &quot;Sezione notizie&quot; o &quot;Sezione viaggi&quot;.

   Consulta [Informazioni sulle raccolte](c-about-settings-menu/c-about-searching-menu.md#concept_62E42ACE53D54EEE9273433B86259127).
* Personalizza la ricerca per l&#39;uso con i set di frame.

   Vedere [Uso dei frame con forms](c-appendices/c-searchforms.md#reference_82CDDDA1E37042E4849EBF7EA05407C5).
* Controlla le posizioni da cui viene avviata una ricerca.

   Consultare [Informazioni sulle restrizioni](c-about-settings-menu/c-about-searching-menu.md#concept_B5B527E04EBF4E9AB5956EEF881DDBF1).
* Specifica i valori utilizzati per testare internamente i modelli.

   Consultare [Informazioni sull&#39;anteprima](c-about-settings-menu/c-about-searching-menu.md#concept_DF293FD3B02C467F8842C8C21D62F294).
* Le opzioni metadati consentono di personalizzare la pertinenza dei campi HTML e metadati (ad esempio le parole chiave o il corpo del documento) che vengono presi in considerazione quando un cliente invia una query di ricerca. Puoi personalizzare e definire i campi di metadati.

   Vedere [Informazioni sulle definizioni](c-about-settings-menu/c-about-metadata-menu.md#concept_AE48035C210145169BE067D396975620).
* Modificare il contenuto dei campi predefiniti. Ad esempio, puoi aggiungere nuovi contenuti, ad esempio &quot;On Sale Now!&quot;) ai meta tag `desc` di una pagina o di un gruppo di pagine. In alternativa, è possibile rimuovere contenuti irrilevanti senza modificare le pagine web effettive.

   Vedere [Informazioni sulle iniezioni](c-about-settings-menu/c-about-metadata-menu.md#concept_DA091920671948A0A893A26B3A2FAAE5).

* Utilizza le opzioni in **[!UICONTROL Settings]** > **[!UICONTROL Filtering]** e **[!UICONTROL Settings]** > **[!UICONTROL Rewrite Rules]** per &quot;riscrivere&quot; il contenuto della pagina prima che sia indicizzato.

* Configura i dizionari per consentire di specificare gruppi di parole correlate (ad esempio, acquisto, acquisto e procura). Queste parole correlate aiutano a restituire risultati rilevanti anche quando la query di ricerca di un cliente non corrisponde esattamente alla terminologia utilizzata nelle pagine web. Con il sinonimo utilizzato nell&#39;esempio precedente, la query di ricerca del cliente &quot;purchase&quot; restituisce pagine contenenti la parola &quot;buy&quot;.

   Vedere [Informazioni sui dizionari](c-about-linguistics-menu/c-about-dictionaries.md#concept_B8028B71EC8144669614C64578EDB034).
